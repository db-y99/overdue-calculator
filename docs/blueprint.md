# **App Name**: Overdue Amount Calculator

## Core Features:

- Input Fields: Text input fields for entering the due amount and the number of overdue days.
- Calculation Engine: Calculates the overdue amount based on the formula provided: (Due Amount * 0.01099) * 1.5 * Overdue Days
- Results Display: Clearly displays the calculated overdue amount, detailing the daily average and daily overdue amounts.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) for a professional and trustworthy feel.
- Background color: Very light indigo (#E8EAF6), a muted tint of the primary hue.
- Accent color: Soft violet (#8E24AA), analogous to indigo, for highlighting results and key actions.
- Font pairing: 'Space Grotesk' for headlines and 'Inter' for body text, providing a modern and readable interface.
- Simple and clean layout with input fields at the top, calculation details in the middle, and final results displayed prominently at the bottom.
- Subtle animations on calculation and results display for a smooth user experience.
